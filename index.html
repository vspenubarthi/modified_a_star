A* Project Overview

Here is my A* Project. This page will specifically describe how to interact with the 49x49 square grid. For information about the inspiration for the project, please visit the GitHub Page, which describes the ideology behind this.

There are several ways to interact with this demo. First, if you cannot see the grid well, feel free to zoom in. Below is a list of different things you can do to interact with the demo above:

  • If you drag your mouse on the grid you will see black blocks appear; these are obstacles. These are like buildings in the real world. The white blocks are open roads that the car can travel on.
  • If you drag your mouse or click on an obstacle, it will disappear.
  • You can set the (x,y) coordinates for the start and end blocks of each pair in the boxes (make sure values <= 49), with the coordinates being calculated as distance from the top-left corner.
  • You can also set the max travel time (in unit squares) for each start/end pair as well; please refer to the link about to find out why this is here.
  • Once you set these, you can hit the Calculate Distance button to calculate the distance between the points and update the grid. The distance will also be written below in unit squares.
  • Finally, you can clear all obstacles with the Clear button and view random placements with the Random button as well.
Once again, feel free to refer to the GitHub Page linked above or contact me for further explanation.
